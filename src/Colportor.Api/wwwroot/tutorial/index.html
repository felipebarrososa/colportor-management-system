O que é o Colportor ID?

É um sistema para ajudar líderes e colportores a organizar o trabalho missionário. Ele facilita o registro de visitas, o acompanhamento de contatos e a gestão de equipes.

Como funciona?

O sistema tem duas partes principais:

  • Para Líderes: Você gerencia seus colportores, acompanha o progresso, e organiza os contatos missionários.
  • Para Colportores: Você tem uma "carteira digital" com seu perfil, status e informações importantes.
